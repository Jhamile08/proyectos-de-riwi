Posted by Alejo Duque on Monday, October 23 2023

All modern websites and web aplications are built using three fundamental technologies: HTML, CSS and JavaScript. These are the languages of the web that we will learn in Riwi

In this post, let's focus on HTML. We will learn what HTML is all about, and why you too should learn it.

What is HTML?

HTML stands for Hyper Text Markup Language. It's a markup language that web developers use to structure and describe the content of a webpage (not a programming language)

HTML consist of elements that describe different types of content: paragraphs, links, headings, images, video, etc. Web browsers understand HTML and HTML code as websites.

In HTML, each element is made up of 3 parts:

  1. The opening tag
  2. The closing tag
  3. The actual element

You can learn more at MDN Web Docs.
